## Run locally

First, clone the repo:
```
git clone < repo-link >
```
Install the packages:
```
npm install
```
Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Navigate to 'http://localhost:3000' in your browser
```bash
http://localhost:3000
```
